La API de aplicaciones de Trophy es un conjunto de endpoints que proporcionan interfaces simples para crear experiencias de gamificación y cubre toda la gama de acciones que un usuario final de tu aplicación puede realizar.

No incluye funciones administrativas globales ni acciones que solo deberían ser realizadas por tus sistemas internos. Si buscas funcionalidad administrativa, consulta la [API de administración](/es/admin-api/introduction).

La API de aplicaciones es accesible a través de [SDKs](/es/api-reference/client-libraries) disponibles en la mayoría de los lenguajes de programación principales o, para quienes deseen gestionar sus propios clientes HTTP, está disponible en la siguiente URL base.

```bash Base URL
https://api.trophy.so/v1
```
