Trophy tiene soporte para varios webhooks que se activan según eventos clave como `achievement.completed`, `leaderboard.finished` y `streak.lost`.

Estos eventos te permiten activar código personalizado y potenciar experiencias de gamificación extendidas usando datos de Trophy.

Todos los webhooks de Trophy se entregan mediante HTTP, lo que permite a los clientes suscribirse para recibir datos de Trophy a través de solicitudes HTTP `POST` regulares.

Lee más sobre cómo empezar con webhooks a continuación.

## Casos de Uso Comunes {#common-use-cases}

Los casos de uso comunes incluyen:

- Activar secuencias personalizadas de emails y notificaciones push
- Almacenamiento de datos
- Potenciar flujos de trabajo no relacionados con gamificación como actualizaciones de suscripción y gestión de funcionalidades.
- Integración con plataformas IFTTT
